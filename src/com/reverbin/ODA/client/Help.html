ODA
Online Disassembler

Introduction

The Online Disassembler (ODA) is a web-based tool for disassembling binaries targeting a wide range of processing platforms. ODA currently supports the following platforms:

  • ARM
  • Intel x86
  • MIPS
  • PowerPC

Work Flow

Binaries can be loaded via file upload or manually created in the Hex tab. The Strings tab displays ASCII strings found in the loaded binary. The Assembly tab displays the disassembly of the currently loaded binary according to the platform, endianness, and base address selected.

Examples

Several example binaries are show in the Examples menu to get you started.
